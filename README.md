# ai_search_comparison_tool
Collection of scripts for AI content retrieval analysis.

## Prerequisites
Create `.env` file and populate it with the following environment variables:
```
OPENAI_API_KEY=<placeholder>
GEMINI_API_KEY=<placeholder>
PERPLEXITY_API_KEY=<placeholder>
```

## Setup and run the script
```
# Create and start python virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install requirements
pip3 install -r requirements.txt

# Run the script
python3 <script>.py
```
